# AI 취업 코치 — 경쟁 분석 & 고도화 전략

> 2026-06 기준. 기존 취업/면접 서비스를 벤치마킹해 **차별점**과 **다음 고도화 로드맵**을 정리.

---

## 1. 경쟁 서비스 지형

| 구분 | 서비스 | 강점 | 약점(우리 기회) |
|---|---|---|---|
| 채용 플랫폼 | **사람인·잡코리아** | 방대한 공고, AI 자소서 코칭, 6페르소나 AI 모의면접 | 기업을 **공시·재무로 깊이 분석은 안 함**, 유료 비중, 공고 위주 |
| 자소서 | **자소설닷컴** | 공고 연동 문항, 글자수·맞춤법, 친구 첨삭 | 면접/기업분석 약함 |
| AI 영상면접 | **뷰인터·잡다(JOBDA)** | 표정·시선·음성 분석, 직무적합도 평가, 기출 | 실제 채용사 평가용에 가깝고, **개인 무료 연습/기업 리서치 통합은 약함** |
| 해외 | **Final Round AI·Huru** | 모의면접 점수·speech clarity·filler·바디랭귀지, 실시간 코파일럿 | 한국 기업·DART·한국어 면접 맥락 없음, 유료 |

### 공통적으로 **아무도 잘 안 하는 빈틈** = 우리 기회
1. **출처 기반 정직한 기업 분석**(DART 사업보고서·재무·공시)을 면접/자소서 준비와 **한 흐름**으로 연결.
2. **무료 + 한국어 + 개인 맞춤**(내 경험·스펙·자소서를 RAG로 재활용).
3. **준비 전 과정 통합**: 기업분석 → 직무탐색 → 자소서 → 모의면접 → 공고 → 보드(현황관리).

---

## 2. 우리의 현재 차별점 (이미 구현)

- 🇰🇷 **DART 기반 기업 심층분석 + AI 요약**: "이 회사가 무슨 사업을 하는지"를 사업보고서 본문으로 설명 — 경쟁사 대비 독보적.
- 🎤 **무료 음성 모의면접 + 6차원 목소리 분석**(떨림·자신감·속도·명료도·충분함·말끝) + **5종 면접관 페르소나** + **꼬리질문** + **세션 리포트**.
- 📄 **내 자소서·스펙 기반 예상 질문** + **RAG**(저장한 경험·스펙을 자소서/면접에 자동 반영·검색).
- 🤝 **정직성**: 데이터 출처 표기, 없는 사실은 지어내지 않음, 거짓 경력 자소서 거부.
- 🧭 **통합 대시보드**(지원현황·일정 캘린더·스펙·자료실).

## 3. 추가/보완 로드맵 (우선순위)

### 🔴 P1 — 임팩트 큰 차별화
- [ ] **표정·시선 분석(웹캠)**: MediaPipe FaceMesh(브라우저)로 시선 안정성·미소·고개 움직임 → 뷰인터식 비언어 피드백을 **무료·온디바이스**로. (개인정보: 영상은 서버 전송 없이 브라우저에서만 처리 → 정직성과 결).
- [ ] **기업별 기출/예상 질문 DB**: DART 사업보고서·뉴스에서 회사 이슈를 뽑아 **회사 맞춤 압박 질문** 자동 생성(이미 일부 구현 → 심화).
- [ ] **자소서 ↔ 면접 연결**: 작성한 자소서를 바탕으로 면접 질문 생성(사람인 벤치마킹) — 이미 'from_materials' 있음 → 자소서 직접 선택 연동.

### 🟡 P2 — 완성도
- [ ] **자소서 고급 도구**: 글자수 실시간(완료)·맞춤법·문장 다듬기·소제목 추천·AI 표현 탐지 회피 가이드.
- [ ] **답변 모범답안 생성**: 내 답변 → STAR 기반 모범답안 재구성 제시.
- [ ] **면접 난이도 단계**(보통/실전/압박) 슬라이더 + 시간 제한 모드(실전 긴장 훈련).
- [ ] **진행률·성장 그래프**: 모의면접 점수 추이, 준비 완성도 게이지.

### 🟢 P3 — 데이터·배포
- [ ] **채용공고**: 사람인 Open API 승인 시 연결(개인 가능). work24는 기업회원 전용이라 보류.
- [ ] **HF Spaces 배포**(LLM_PROVIDER=gemini) + 스크린샷 + 포트폴리오 통합.
- [ ] 직업훈련·자격증 추천(고용24 추가 API).

## 4. 한 줄 포지셔닝

> **"공시로 회사를 읽고(DART), 내 경험으로 자소서를 쓰고(RAG), 목소리로 면접을 연습하는(음성·페르소나) — 무료·한국어·정직한 취업 올인원."**

기존 서비스가 *공고(사람인)* 또는 *영상평가(뷰인터)* 또는 *해외·유료(Final Round)* 에 머무는 사이,
우리는 **기업 데이터 + 개인 데이터 + 면접 코칭을 정직하게 통합**한다.
